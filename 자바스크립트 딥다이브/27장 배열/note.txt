유사배열 객체 : 배열처럼 인덱스로 접근할 수 있으며 length프로퍼티를 갖는 객체
    arr = {
        '0' : 1,
        '1' : 2,
        length : 2
    }

희소배열이란 length 값과 요소의 개수가 다른 배열이다.

배열의 요소는 사실 객체의 프로퍼티이다.
    arr[0] = 1 arr[1] = 2    =>
    arr = {
        '0' : 1,
        '1' : 2,
        length:2
    }

배열도 객체이기 때문에 프로퍼티를 동적으로 추가할 수 있다.
    예) arr = []
        arr.one = 1
        arr["two"] = 2

동적으로 추가한 숫자인 인덱스가 아닌 프로퍼티는 length에 영향을 주지 않는다.
