데이터 타입
- 값의 종류
- 자바스크립트의 모든 값은 데이터 타입을 갖는다.
- 총 7개의 데이터타입이 있음.
7개의 데이터타입은 원시 타입과 객체 타입으로 분류
- 동적 타이핑
데이터 타입을 계속해서 변경 가능



* 원시타입(6개) + 객체타입(1개)
원시타입
- 숫자 타입
  모든 수를 실수로 처리

- 문자 타입
- 불리언 타입
- undefined 타입
var로 선언한 변수는 암묵적으로 undefined로 초기화됨.
값이 있긴 하지만 정의되지 않았다는 것을 의미
- null 타입
값이 없다.
- symbol 타입


객체 타입
- 자바스크립트를 이루고 있는 거의 모든 것이 객체


데이터 타입의 필요성
- 메모리 공간의 확보와 참조
각 데이터 타입에 따라 메모리 공간이 다르며 해당 값이 무엇인지 구분하게 해줌.
